OK Kinderhook

A Village Loyalty Program
~$1,500
to launch / free to run
The simple idea: Make it easy and rewarding for residents to shop at local businesses. No complicated technology. No cost to businesses. Just a membership card, some friendly competition, and a way to keep dollars circulating in our village.

How It Works

1. Member Discounts

Every village resident gets an OK Member card. Show it at participating businesses for exclusive discounts.

Like a AAA card - but for Kinderhook.

2. Visit Tracking

Each visit gets recorded. You can see which local shops you've supported this week, this month, this year.

Like a punch card that never gets lost.

3. Earn Rewards

Visit regularly and earn badges. Complete fun challenges - like visiting 5 shops in a week - and earn bonuses.

Like frequent flyer miles for Main Street.

4. Kinderbucks

Physical local currency accepted at participating businesses. Special deals for Kinderbucks only.

Like BerkShares in the Berkshires.

For Businesses

  • Completely free to participate
  • Set your own discount level
  • See which customers visit most
  • Run joint promotions with neighbors
  • Get listed in the village directory

For Residents

  • Free membership for all villagers
  • Discounts at local shops & restaurants
  • Fun challenges and rewards
  • Trade badges for real prizes
  • Support the neighbors you know

No Smartphone? No Problem.

Your physical OK Member card works everywhere. The phone features are optional extras.
