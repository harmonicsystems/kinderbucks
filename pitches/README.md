## Suggested Pitch Flow

**For Business Owners:** Start with `PRINTABLE` (simple), have `VILLAGE-OS` ready for deeper questions

**For Library:** Start with `COOPERATIVE` (emphasizes learning track), show `VILLAGE-OS` for the bundles

**For Village Board:** Lead with `VILLAGE-OS`, use `PITCH.md` for talking points

**For Quick Elevator Pitch:** Use `SLIDE-SIMPLE` on your phone
